Getting Started

To start the Flask server, follow these steps:

  1. Open your terminal.
  2. Navigate to the project directory: 
    cd /path_to_the_repo/backend
  3. Create python virtual environment: 
    python -m venv .venv
  4. Activate your virtual environment: 
    source .venv/bin/activate
  5. Note: Please ensure that your environment is set up with all necessary dependencies. Use the following command to install any required packages: 
    pip install -r requirements.txt
  6. Run the Flask application: 
    python main.py
  7. The server will start, and you can access the API at: 
    http://127.0.0.1:5000

Authentication Endpoints

1. Register User

Endpoint: /register

Method: POST

Request Body:

{
  "username": "johndoe",
  "email": "john@example.com",
  "password": "yourpassword",
  "firstName": "John",
  "lastName": "Doe"
}

Response:

{
  "message": "User registered successfully",
  "access_token": "jwt_token_here"
}

2. Login

Endpoint: /login

Method: POST

Request Body:

{
  "username": "johndoe",
  "password": "yourpassword"
}

Response:

{
  "access_token": "jwt_token_here"
}

3. Logout

Endpoint: /logout

Method: POST

Authentication: Required

Response:

{
  "message": "Successfully logged out"
}

Post Management

1. Create Post

Endpoint: /create_post

Method: POST

Authentication: Required

Request Body:

{
  "title": "My First Post",
  "body": "This is the body of my first post.",
  "tags": ["technology", "programming"]
}

2. Get Single Post

Endpoint: /get_post/{post_id}

Method: GET

Authentication: Optional

3. Get All Posts

Endpoint: /get_posts

Method: GET

Query Parameters:

  • page (optional): Page number
  • tag (optional): Filter by tag

4. Edit Post

Endpoint: /edit_post/{post_id}

Method: PUT

Authentication: Required

Request Body:

{
  "title": "Updated Title",
  "body": "Updated content"
}

5. Delete Post

Endpoint: /delete_post/{post_id}

Method: DELETE

Authentication: Required
