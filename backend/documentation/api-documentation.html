Getting Started

To start the Flask server, follow these steps:

  1. Open your terminal.
  2. Navigate to the project directory: 
    cd /path_to_the_repo/backend
  3. Note: Please ensure that your environment is set up with all necessary dependencies. Use the following command to install any required packages: 
    pip install -r requirements.txt
  4. Activate your virtual environment: 
    source .venv/bin/activate
  5. Run the Flask application: 
    python main.py
  6. The server will start, and you can access the API by default at (check the terminal for more). 
    http://127.0.0.1:5000

Available Endpoints

1. Create Post

Endpoint: /create_post

Method: POST

Request Body: All three fields (hostId, title, body) are required. 

{
  "hostId": 1,
  "title": "My First Post",
  "body": "This is the body of my first post."
}

Response:

{
  "message": "Post created successfully"
}
